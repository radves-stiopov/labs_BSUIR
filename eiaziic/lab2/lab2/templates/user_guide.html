User Guide

This web application allows you to manage articles and words, upload article content, and analyze word data. Follow the guide below to get started:

1. Viewing Articles

  • Click the **"View Articles"** button on the main page to see a list of all articles.
  • From the articles page, you can:
    • View details of a specific article by clicking on it.

2. Viewing Words

  • Click the **"View Words"** button on the main page to see a list of all words and their counts.
  • Use the search bar at the top of the words page to filter words by name.

3. Uploading a File

  • To upload a file, use the **"Upload a File"** section on the main page.
  • Click the **"Choose File"** button to select a `.txt` file from your computer.
  • Once a file is selected, click the **"Upload File"** button to upload and process the file.
  • After processing, flash messages will display the status of the upload.
  • Ensure the file is in `.txt` format and contains valid data to avoid errors.

4. Updating Part-of-Speech (POS) Information

  • While viewing an article's details, you will see a list of words associated with the article.
  • Each word has a **POS Info** field that can be updated by submitting the corresponding form.
  • Provide the updated POS Info, and the system will save the changes in the database.
